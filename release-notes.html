Release Notes

Version history for ExcelLink for Revit

Version 1.0
Initial Release March 2026

New

  • Export Revit element parameters to .xlsx with per-category or single-sheet output.
  • Import updated values back to Revit in a single transaction using element Unique IDs for matching.
  • Support for Revit 2022, 2023, 2024, 2025, and 2026.
  • Linked document support — export parameters from loaded Revit link instances.
  • Include or exclude type parameters on a per-export basis.
  • Drag-and-drop parameter column reordering before export.
  • Save and load export configuration to a .json file for reuse.
  • Cell color coding: white (writable), pink (read-only), yellow (type), red (missing), gray (linked).
  • Trial mode with 10-element export limit per category — no registration required.
  • Online license activation with 30-day offline grace period.
  • Shared parameter matching via GUID; built-in parameter matching via internal ID — reliable across model transfers.
  • Double values always exported with InvariantCulture decimal format (period separator) regardless of system locale.
  • Floating-point precision rounding to 6 decimal places eliminates unit conversion noise (e.g. 6920 mm roundtrips cleanly).
