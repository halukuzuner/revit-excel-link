Section 01

Introduction

This Privacy Policy describes how Haluk Uzuner ("Publisher", "we", "us") collects, uses, stores, and protects information when you use the ExcelLink for Revit add-in ("the App").

By installing or using the App, you agree to the practices described in this policy. This policy applies to all versions of ExcelLink for Revit compatible with Autodesk Revit 2022 through 2026.

Section 02

Data We Collect

The App collects only the minimum data necessary to validate your license and manage active sessions. No personally identifiable information beyond what is listed below is collected.

Data Purpose Retention
License key License validation and session management Until license expiry date
Hardware identifier Binding the license to the registered device Until license expiry date
Computer name Identifying the active session in the admin interface 45 days from last session activity
Session token Maintaining an authenticated active session 45 days from last session activity
Customer email address License issuance and support communication Until license expiry date

Important: The App does not collect, transmit, or store any Revit model data, project files, parameter values, or user-generated content of any kind. No IP addresses are logged or stored.

Section 03

How We Use This Data

All collected data is used exclusively for the following purposes:

  • Verifying that a valid license exists for the installed device
  • Enforcing concurrent session limits as defined by the license type
  • Enabling the offline grace period that allows the App to operate without an internet connection for a limited time
  • Providing license management capabilities to the publisher via a private administrative interface

We do not use your data for advertising, analytics, profiling, or any purpose other than those stated above.

Section 04

Data Sharing and Third Parties

We do not sell, rent, trade, or otherwise share your data with any third party. The license validation server is operated solely by the publisher.

No third-party analytics platforms, advertising networks, or data brokers have access to any data collected by this App.

Section 05

Data Security

We take reasonable technical measures to protect your data:

  • All communication between the App and the license server is encrypted using HTTPS (TLS)
  • Session tokens are randomly generated cryptographic values
  • The license database is stored on a private server accessible only to the publisher
Section 06

Data Retention and Deletion

Data is retained only for as long as necessary for the purposes described in this policy:

  • License records (license key, hardware identifier, email address) are retained until the license expiry date and deleted thereafter.
  • Session records (session token, computer name) are automatically deleted 45 days after the last recorded session activity.

You may request deletion of your data at any time by contacting us as described in Section 9. Upon a valid deletion request, all data associated with your license will be permanently removed within 30 days.

Section 07

Your Rights

You have the right to:

  • Request a copy of the data we hold about you
  • Request correction of inaccurate data
  • Request deletion of your data
  • Withdraw consent by ceasing use of the App and requesting data deletion

To exercise any of these rights, please contact us using the information in Section 9.

Section 08

Changes to This Policy

We may update this Privacy Policy from time to time. When we do, the updated version will be published at www.halukuzuner.com.tr/privacy-policy with a revised effective date.

Continued use of the App after any changes constitutes acceptance of the updated policy.
